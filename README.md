# Smart Conversational Enrollment Agent

## Overview
A state‑driven conversational AI agent that assists users in course enrollment through a natural multi‑turn interaction.

## Features
- Multi‑turn conversational flow
- Graceful handling of corrections and missing data
- External API integration for enrollment
- Zero‑installation, browser‑based execution

## How to Run
Open `index.html` in any modern web browser.

## Demo Flow
User → Course inquiry → Details capture → Correction → Enrollment → Confirmation

## External Tool
Enrollment data is sent to an external webhook API.
